[USAGE IN TESTBENCH]
====================

The aes256_nist_validation_tb.sv automatically:
1. Loads these test vectors
2. Encrypts plaintext with given key
3. Compares output to expected ciphertext
4. Reports PASS/FAIL for each vector
5. Performs round-trip verification (encrypt → decrypt)
6. Generates CAVP validation report
